# CareerAI Job Board

An AI-powered job board built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Features

- AI Resume Match Analyzer
- AI Cover Letter Generator
- AI Interview Preparation
- Job Search & Filtering
- Saved Jobs
- Dashboard Analytics
- Responsive Design
- Dark Mode
- GitHub Actions CI/CD
- Vercel Deployment

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Recharts
- React Router DOM

## Getting Started

```bash
npm install
npm run dev
